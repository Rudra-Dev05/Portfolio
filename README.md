### Theme Configuration

Colors can be customized in the `tailwind.config.mjs` file:

```js
// Light mode, dark mode, and accent colors
colors: {
  cream: { /* Light mode palette */ },
  midnight: { /* Dark mode palette */ },
  accent: { /* Highlight colors */ }
}
```
